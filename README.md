# Woosan-front
우산 프로젝트 프론트입니다

## 세팅환경
기본환경 : React + TypeScript + Vite

# 기본 시작 명령어
npm run dev

# 파일구조
```
woosan-front
├─ eslint.config.js
├─ index.html
├─ package.json
├─ public
│  └─ vite.svg
├─ README.md
├─ src
│  ├─ app
│  ├─ App.css
│  ├─ App.tsx
│  ├─ assets
│  │  └─ react.svg
│  ├─ components // 공통적으로 사용되는 함수모임
│  ├─ constants // 공통적으로 사용되는 상수모임
│  ├─ css // css 파일
│  ├─ data // 더미데이터
│  ├─ hooks
│  ├─ index.css
│  ├─ lib
│  ├─ main.tsx
│  ├─ types
│  ├─ utils
│  └─ vite-env.d.ts
├─ tsconfig.app.json
├─ tsconfig.json
├─ tsconfig.node.json
└─ vite.config.ts

```
